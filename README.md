# 关于 IPA 表 和 字母表

这是一个纯享极简版的音标工具,没有广告、复杂的操作界面,上手即用,无论是学习还是查询都非常的方便.

## 使用方式:

直接点击音标即可听其发音.

IPA 音标配有示例单词,帮助您理解其在语境中的用法.

字母表采用周育如老师的发音作为参考,其采用的是 KK 音标发音法

## 什么是 IPA?

国际音标（IPA）是由国际语音学协会开发的标准化语音记号系统。

它使用一套符号来表示口语的声音，使其成为语言学习者、教师和语言学家的宝贵工具。

## 注意:

这只是一个粗略的指南。根据语言和上下文的不同，这些声音的发音会有很多变化。

例如，在英语中，清辅音爆破音通常以一股称为送气的气流结束，但此页面上的清辅音爆破音没有送气。

aeiou 发母音(17 个),其他字母发子音,子音绝大多数情况发音不变

1. 一个单词里有 2 个元音字母的情况下,第一个元音字母发重音节，然后第二个元音字母绝大多数发`/ə/`音

2. 在一个单词中有 2 个元音字母，且结尾字母是元音字母 `e`，那么这个 `e` 不发音，第一个元音字母发自身的音

3. `ar`/`er`/`ir`/`or`/`ur`,在单词中第一次出现通常发`/3:/`音，在结尾出现发`/ə/`音

4. 单词里两个 `o` 相连，大概率发`/ʊ/`,如 `book` /`bʊk`/,小概率发`/u:/`，如 `room`: `/rum/`

5. `au`/`ou`/`aw`/`ow` 常常发音为`/aʊ/`,如 `now`: `/naʊ/`
